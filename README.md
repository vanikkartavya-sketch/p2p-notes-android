# P2P Notes 📱📚

A premium, high-fidelity peer-to-peer academic notes platform designed for students to seamlessly upload, share, and manage study resources. Built fully with **Jetpack Compose** and **Material 3**, this interface follows minimalist design principles with a sleek dark theme.

## ✨ Features
* **Slate Navy Canvas:** Implemented a modern, deep-slate dark mode for a professional, high-end visual look.
* **Polished Profile Dashboard:** Complete with dynamic profile tags, role capsules ("STUDENT"), and customizable student avatars.
* **Business Ledger & Settlement:** Real-time earnings tracking panel (supports view scaling with dynamic balance updates) along with instant simulated UPI/bank settlement.
* **Unified Workspace:** Profile settings, account workflows, and historic purchased documents are consolidated cleanly into borderless unified container blocks.
* **Sleek Bottom Navigation:** Clean navigation bar featuring low-profile custom indicators for quick tab switching.

## 🛠️ Tech Stack
* **Language:** Kotlin
* **UI Framework:** Jetpack Compose (Material 3 Design System)
* **Architecture:** MVVM Pattern
